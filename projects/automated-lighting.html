Automated Lighting System

Designed a custom PCB using LM358-based relay switching circuitry for intelligent automated lighting control with ambient light sensing, manual override, and energy-efficient operation.

📋 Project Overview

This project involved designing and manufacturing a custom PCB for automated lighting control. The system uses an LM358 op-amp as a comparator to detect ambient light levels and automatically switches lights on/off via relay modules. The design prioritizes energy efficiency and reliability.

⚡ Key Features

  • Custom PCB Design — Designed from scratch using KiCad with proper grounding and trace routing
  • LM358 Comparator Circuit — Precision ambient light detection using LDR + op-amp configuration
  • Relay Switching — High-current relay module for safe switching of AC/DC loads
  • Manual Override — Physical switch for manual control when automation isn't desired
  • Energy Efficient — Low quiescent current draw in standby mode
  • Status LEDs — Visual indicators for system state and relay status
